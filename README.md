# 公众号文章生成器

这是一个纯前端的小工具，用于输入文章标题和内容概述后，生成公众号文章预览、配图建议与排版提示，并支持二次修订。

## 快速启动

> 推荐方式（本地启动一个静态服务器）：

```bash
python -m http.server 8000
```

然后在浏览器打开：

```
http://localhost:8000/index.html
```

> 也可以直接双击打开 `index.html`，但部分浏览器在本地文件模式下会限制脚本或字体表现，建议使用本地静态服务器。

## 文件说明

- `index.html`：页面结构与交互布局。
- `styles.css`：页面样式与响应式布局。
- `app.js`：生成文章、配图和修订逻辑。
